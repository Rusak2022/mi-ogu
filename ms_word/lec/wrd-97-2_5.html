2.5 Отмена действий

  1. Для вывода списка действий, которые могут быть отменены, нажмите стрелку рядом с кнопкой Отменить   на панели быстрого доступа (рядом с кнопкой Office).
  2. Щелкните действие, которое следует отменить. Возможно, для его поиска потребуется прокрутить список.
  3. Для повторного выполнения последнего действия нажмите кнопку Повторить на панели быстрого доступа.
  4. Для повторного выполнения последнего отмененного действия нажмите кнопку Вернуть на панели быстрого доступа.
