VII. ОФОРМЛЕНИЕ ДОКУМЕНТА

7.1 Разделы документа

Иногда необходимо различные части документа оформить различным образом. Например, в документ, основная часть которого набрана для обычной ориентации страницы "Портрет" нужно вставить несколько широких таблиц и для этого требуется развернуть несколько страниц на 90°.

Эту задачу можно решить по-разному. Можно, например, создать новый документ с другой ориентацией страниц. Однако, удобнее в одном документе создать несколько различным образом оформленных разделов.

Каждый из разделов может иметь свои поля, ориентацию страницы, колонтитулы (см. ниже) и т.д.

Для вставки нового раздела выполните следующие действия:

  1. Выберите в меню Вставка пункт Разрыв.
  2. В открывшемся диалоговом окне в группе Новый раздел выберите один из следующих пунктов:
    • Со следующей страницы;
    • На текущей странице;
    • С четной страницы;
    • С нечетной страницы.
