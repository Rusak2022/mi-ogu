ЛАБОРАТОРНАЯ РАБОТА № WRD-02

ИСПОЛЬЗОВАНИЕ СТИЛЕЙ В MS WORD

Теоретические сведения:

  1. Назначение текстового процессора Word.
  2. Понятие стиля абзаца. Использование стилей.
  3. Создание оглавления.

Задания:

  1. Загрузите документ B-1,  B-2,  B-3,  B-4,  B-5,  B-6,  B-7,  B-8,  B-9,  B-10,  B-11,  B-12,  B-13,  B-14,  B-15.  Сохраните его копию в своей сетевой папке. Имя копии — Word-02.
  2. Установите размер бумаги — A4.
  3. Настройте поля: верхнее, правое и нижнее — по 2 см, левое — 2,5 см.
  4. Настройте стили абзацев:
              Создайте стиль с именем "[Ваша фамилия]-1"
    5. (если за компьютером сидит несколько студентов, то в названии стиля перечисляете все фамилии;
    вкладка Главная -> Стили -> открыть окно стилей (кнопка ) -> кнопка ),
    основанный на стиле Обычный, с параметрами:
    • левая и правая границы — по полям;
    • абзацный отступ— 1.25 см;
    • выравнивание — по ширине;
    • междустрочный интервал — одинарный;
    • шрифт — Times New Roman, размер 12 пт.

      Создайте стиль с именем "[Ваша фамилия]-2",
      • основанный на стиле Заголовок1, с параметрами:
    • левая и правая границы — по полям;
    • абзацный отступ — 0;
    • выравнивание — по центру;
    • шрифт — Arial; размер 14 пт, полужирный, заглавные буквы.
    • заголовки должны автоматически нумероваться со значком § впереди (в диалоге создания стиля выберите кнопку Формат -> Нумерация -> Определить новый формат номера -> Вставьте значок перед номером в поле "формат номера").
    • настройте положение заголовка параграфа - автоматически с новой страницы.
  5. Оформите основной текст документа стилем "[Ваша фамилия]-1" заголовки параграфов - стилем "[Ваша фамилия]-2".
  6. Пронумеруйте страницы документа. Отформатируйте номера страниц:
    • номер страницы расположен снизу, центрирован;
    • размер шрифта номера страницы — 11 пт.
  7. На отдельной странице в конце документа составьте его оглавление (вкладка Ссылки -> кнопка Оглавление -> (шаблоны не берем, т.к. они не всегда правильно работают с пользовательскими стилями), снова выбираем пункт Оглавление -> кнопка Параметры -> ищем свои стили, проверяем: напротив стиля текста не должно быть уровня, напротив стиля заголовка - 1 уровень, если он один - например, только главы).
  8. Оформите титульный лист документа, перенеся на него название документа, имя и данные автора.
