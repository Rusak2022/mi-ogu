Создание тестового документа

  1. Перейдите в ту папку, внутри которой должна быть создан новый текстовый документ.
  2. Щелкните на свободном месте правой кнопкой мыши.
  3. Из контекстного меню выберите команду Создать и подкоманду Текстовый документ:

    4.  

  4. На экране появится новый текстовый документ с именем "Текстовый документ":

  5. Введите нужное имя и нажмите ENTER.
