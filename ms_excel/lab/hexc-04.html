ЛАБОРАТОРНАЯ РАБОТА № EXC-04

КОПИРОВАНИЕ ДАННЫХ МЕЖДУ ЛИСТАМИ. ФОРМУЛЫ

Теоретические сведения:

  1. Выделение ячеек, диапазонов, строк и столбцов.
  2. Перемещение и копирование ячеек
  3. Копирование данных внутри строки или столбца
  4. Копирование диапазона в несколько областей
  5. Копирование только видимых ячеек
  6. Раздельное копирование значений, формул, примечаний и форматов ячейки
  7. Ссылки на ячейку или на группу ячеек
  8. Различия между относительными и абсолютными ссылками
  9. Ввод формулы
  10. Синтаксис формулы

Задания:

  1. Загрузите документ LRexc-04 и сохраните его копию в своей сетевой папке под тем же именем.
  2. Удалите все рабочие листы, кроме листов "Численность" и "ВВП". На листе "Численность" находятся данные по численности населения стран СНГ, на листе "ВВП" - данные по ВВП этих же стран.
  3. Просуммируйте данные по численности населения и по ВВП каждой из стран за каждый год и поместите их в строку "СНГ", которую вставьте над строкой "Россия". Таблицы оформите в соответствии со следующими образцами: Образец для "Численность", Образец для "ВВП".
  4. Рассчитайте ВВП на душу населения и поместите их на новый рабочий лист с именем "ДушевойВВП". Для этого данные из каждой ячейки рабочего листа "ВВП" нужно разделить на соотвествующее данное рабочего листа "Численность". Вычисления удобно проводить в следующем порядке:
    • Перейдите на новый рабочий лист "ДушевойВВП".
    • Скопируйте наименования стран и строку с годами в нужное место новой таблицы.
    • В первую ячейку данных создаваемой таблицы введите формулу, позволяющую вычислить ВВП в расчете на душу населения с учетом наименований листов. Например, в ячейке B5 рисунка должна стоять следующая формула: =ВВП!B5/Численность!B4.
    • Скопируйте полученную формулу во все ячейки новой таблицы.
    • Настройте формат чисел, удерживая в дробной части два знака после запятой.
    • Настройте обрамление таблицы, заливку ячеек и оформите заголовок по приведенному ниже образцу.

  6. На новом рабочем листе "Относительный ВВП" постройте таблицу, ячейки которой содержат данные об относительном ВВП для каждой из стран СНГ (по отношению к СНГ в целом). Образец можно посмотреть здесь.
