1.2 Создание книги

  1. Выберите команду Создать в меню Файл.
  2. Чтобы создать новую пустую книгу, выберите вкладку Общие и дважды щелкните значок “Книга”.
  3. Для создания книги по шаблону выберите вкладку со списком стандартных шаблонов или вкладку со списком пользовательских шаблонов, а затем дважды щелкните шаблон, по которому создается книга.
  4. Чтобы создать новую книгу по стандартному шаблону книги, нажмите кнопку “Создать”.
